
# 🔐 Interactive Password Generator App

A modern, responsive, client-side web application that generates secure cryptographic passwords based on customizable criteria. Features a real-time visual strength metric gauge and a streamlined mobile layout.

🌐 **[Live Demo Link](https://nimraaltaf.github.io/Password-Generator-App/)**

---

## 🚀 Features

- **Dynamic Generation Engine:** Custom character pools based on user inputs (Uppercase, Lowercase, Numbers, Symbols).
- **Asynchronous Clipboard Integration:** One-click copy utility utilizing the modern Web Clipboard API with temporary visual UI state feedback.
- **Dynamic SVG Metric Gauge:** Real-time geometric calculation tracking password structural strength using custom formula offsets ($377 - (\text{Score}/100 \times 377)$).
- **Adaptive Architecture:** 100% mobile responsive structural design optimizing desktop grid frameworks into easy-to-use tap targets on mobile viewports.

---

## 🛠️ Tech Stack

- **Structure:** Semantic HTML5
- **Styling:** Modern CSS3 (Flexbox, CSS Grid, Custom Media Queries, SVG Dash-Arrays)
- **Engine:** Vanilla JavaScript (ES6+, Asynchronous Web APIs, Mathematical DOM Mutation)

---

## 🧠 What I Learned / Core Engineering Concepts

- **CSS Specificity Overrides:** Managing ID selector priority hierarchies within viewport breakpoints to enforce responsive style changes.
- **Asynchronous Flow Management:** Handling JavaScript Promises via chained `.then()` methodologies for platform API integration.
- **Compound Assignment String Compilation:** Optimizing runtime memory performance using assignment operators (`+=`) during dynamic string generation iterations.
